# homebrew-tools

Homebrew formula for installing Simon's apps & tools.

## How To Use

Add repository:

```
brew tap simonwehle/tools
```

Install [yt2abs](https://github.com/simonwehle/yt2abs):

```
brew install yt2abs
```

Upgrade the yt2abs CLI to the latest version:

```
brew upgrade yt2abs
```
